PROYECTO RT DESCRIPCION DE USUARIOS Y MANEJO DE APP

LA APP VA A TENER EL NOMBRE DE LA AGENCIA:
RINCON TRAVEL ES UNA AGENCIA DE TURISMO INTERNA DE UNA EMPRESA DE TIEMPO COMPARTIDO QUE SE ENCARGA DE ASESORAR Y BRINDAR SERVICIO A SUS MAS DE 6000 SOCIOS.

RINCON TRAVEL ES LA APP QUE VA A TENER DESPLIEGUE TANTO PARA ANDROID COMO PARA IOS. LA APLICACIÓN VA A TEENR 4 TIPOS DE USUARIOS QUE VAN EXPLICADOS A CONTINUACIÓN
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
USUARIO SOCIO:
ESTE USUARIO TIENE QUE PAAR POR UNA ETAPA DE REGISTRO CON EL ADMINISTRADOR. UNA VEZ APROBADO SU REGISTRO PUEDE PASAR A LA SOLICITUD DE SERVICIOS.
ENTRE SUS OPCIONES TIENE SOLICITUD DE SERVICIOS, PENDIENTES, CONFIRMADOS


USUARIO EMPLEADO: 
ESTE TIENE QUE PASAR POR UNA ETAPA DE REGITRO CON EL SUPER-ADMIN. UNA VEZ APROBADO SU REGISTRO VA A PODER RECIBIR SOLICITUDES DE SERVICIO QUE LE VAN A LLEGAR DEL ADMINISTRADOR.
ENTRE SUS OPCIONES TIENE SOLICITUDES DE SERVICIO, PENDIENTES Y CONFIRMADOS.


USUARIO ADMINISTRADOR:
ESTE TIENE QUE PASAR POR UNA ETAPA DE REGISTRO CON EL SUPER-ADMIN.UNA VEZ APROBADO SU REGISTRO YA PUEDE RECIBIR SOLICITUDES DE SERVICIO Y DERIVARLAS A SUS EMPLEADOS, ADEMAS DE RECIBIR SOLICITUDES
DE REGISTRO DE LOS SOCIOS, VERIFICARLAS Y APROBARLAS. 

USUARIO SUPER-ADMIN:
ESTE USUARIO ES EL ENCARGADO DE APROBAR LOS REGISTROS DE USUARIOS ADMINISTRADORES Y DE USUARIOS EMPLEADOS. TAMBIEN ES EL ENCARGADO DE ELIMINARLOS LLEGADO EL CASO. 

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


SERVICIOS DE LA AGENCIA:

* TRANSFER
* ALQUILER DE VEHICULO
* ESTADIAS
* PASAJES
* EXTRAS (CHAPELCO)
* ASESORAMIENTO


TRANSFER:
PARA PODER BRINDAR ESTE SERVICIO EL EMPLEADO DEBE CONOCER LUGAR DE PARTIDA,CANTIDAD DE PASAJEROS,  DESTINO, FECHA DE IDA, FECHA DE VUELTA , HORA DE LLEGADA Y NUMERO DE VUELO (EN CASO DE VIAJE EN AVION) TANTO DE IDA COMO DE VUELTA.
EL SERVICIO PUEDE SER SOLO DE IDA, SOLO DE VUELTA O DE IDA Y VUELTA. 
 
ALQUILER DE VEHICULO: 
PARA PODER BRINDAR ESTE SERVICIO EL EMPLEADO DEBE SABER FECHA Y HORA PARA BUSCAR EL VEHICULO Y LO MISMO PARA DEJARLO. TAMBIEN CANTIDAD DE PERSONAS. 

ESTADIAS:
ESTE ESPECTRO ES TAN AMPLIO QUE PARA PDER BRINDAR EL SERVICIO EL EMPLEADO DEBE SABER FECHAS ESTIMADAS, CANTIDAD DE PERSONAS Y CIUDAD DE DESTINO. EN BASE A ESTO EL EMPLEADO YA CUENTA CON SUFICIENTE INFORMACIÓN PARA REALIZAR UNA COTIZACIÓN.

PASAJES:
HAY 3 POSIBILIDADES
MICRO, BUQUE Y AVIÓN
PARA PODER COTIZAR SE NECESITA LUGAR DE PARTIDA, LUGAR DE DESTINO, FECHA DE IDA Y FECHA DE VUELTA, CANTIDAD DE PERSONAS, Y EN EL CASO DE BUQUE SI PIENSA LLEVAR VEHICULO (USO DE BODEGA). 

CHAPELCO:
ESTE CASO ES ESPECIFICO DE LA AGENCIA: PASES, ALQUILER DE EQUIPOS, TRANSFER AL CERRO, CLASES
PARA ESTO EL EMPLEADO SOLO NECESITA SABER FECHAS DE IDA Y DE VUELTA, Y LA CANTIDAD DE DIAS QUE PIENSA IR AL CERRO.
EL RESTO DEPENDE DEL EMPLEADO EN COMUNICARSE Y COTIZAR.

ASESORAMIENTO:
ESTA OPCION VA A SER UNA SIMPLE CAJA DE MENSAJE DE MAXIMO 500 PALABRAS.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

CONFIRMACIÓN DE SOLICITUD:
EL EMPLEADO AL COMUNICARSE CON EL SOCIO Y COTIZARLE, SI LOGRA LA VENTA VA A DEJAR EL MONTO QUE LE COBRO. ESTA CONFIRMACIÓN LE VA A LLEGAR AL ADMINISTRADOR Y AL SOCIO. ADEMAS DEBE QUEDAR REGISTRADA JUNTO CON EL MONTO EN UNA BASE DE DATOS APARTE PARA PODER FACTURARLE A LA EMPRESA A FIN DE MES.
HASTA QUE ESTA CONFIRMACIÓN LLEGUE, DEBE QUEDAR Y FIGURAR COMO PENDIENTE TANTO PARA EL SOCIO COMO PARA EL ADMINISTRADOR Y AL EMPLEADO.
